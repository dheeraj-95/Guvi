Heading h1 represents the main heading/subject for the whole page

Heading h2 represents the sub heading/section

Heading h3 represents the sub sub heading. It can be used as "Leave a Reply" in blogs(With font-size 1.17em in chrome)

Heading h4 represents level 4 heading in an HTML document.(With font-size 1.33em in chrome)

Heading h5 represents level 5 heading in an HTML document.(With font-size 0.83em in chrome)
Heading h6 tags are used to define heading levels, like in a Table of Contents.(With font-size 0.67em in chrome)
